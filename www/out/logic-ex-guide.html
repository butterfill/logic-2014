Logic-ex user guide

--- by s.butterfill@warwick.ac.uk

Some notes on using logic-ex to complete logic exercises (writing proofs, creating counterexamples, ...).

For students:

Counterexamples

definitions

Explains the notion of a counterexample. Discusses a counterexample to a simple argument. Demonstrates using logic-ex to construct a counterexample involving shape and size properties.

Formal Proof: ∧Elim and ∧Intro

proof

Introduces and illustrates the use of rules of proof for conjunction. Explains how to create proofs using logic-ex.

Identity

definitions

Introduces the notion of identity used in studying first-order logic. Includes example argument and the two principles. Also demonstrates how to assign names in logic-ex, and using logic-ex to make identity statements true and false.

Contradictions, Logical Truths and Logical Possibilities

definitions

Illustrates and defines the terms ‘contradiction’, ‘logical truth’ and ‘logical possibility’. Describes how to recognise these from truth tables. Demonstrates using logic-ex to construct truth-tables and answer questions about contradictions, logical truths and logical possibilities.

Logic-Ex: Creating Possible Situations

Describes how to create possible situations in logic-ex.
